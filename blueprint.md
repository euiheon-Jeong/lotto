# Lotto Number Generator Blueprint

## Overview
A modern, vibrant web application that generates random lotto numbers. Users can click a button to receive 5 sets of 6 unique numbers between 1 and 45.

## Current State & Features
- Initial setup with basic HTML, CSS, and JS structure.
- Responsive design principles (planned).
- Modern aesthetic with interactive elements (planned).

## Implementation Plan
1.  **UI Development (`index.html`):**
    - Create a clean layout with a header, a main action button, and a result container.
    - Use semantic HTML.
2.  **Styling (`style.css`):**
    - Implement a vibrant color palette using CSS variables.
    - Add interactive button styles (hover, active effects).
    - Design "balls" for the lotto numbers with deep shadows and gradients.
    - Ensure mobile responsiveness.
3.  **Logic (`main.js`):**
    - Implement a random number generator function (1-45).
    - Create a loop to generate 5 sets of 6 unique numbers.
    - Dynamically update the DOM to display the results with a slight entry animation.
